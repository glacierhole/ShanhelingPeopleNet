# R语言 基于共现提取《天涯客》人物关系并做网络图


基于简单共现关系，编写 R 代码从《天涯客》小说纯文本中提取出人物关系网络，并用 networkD3 将生成的网络可视化。

* 天涯客中文文本 		天涯客.txt
* 人物名词           人物表.xlsx
* 多称谓人物表       多称谓人物.xlsx
* 主要程序           netgraphicSHL.rmd
* 报告结果           netgraphicSHL.html
